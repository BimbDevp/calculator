# Calculator Web App

Kalkulator web modern, bersih, dan responsif dengan desain **Neumorphism (Soft UI)** yang premium. Proyek ini dibangun dari nol sebagai bagian dari kurikulum Foundations di The Odin Project untuk menunjukkan pemahaman mendalam tentang struktur HTML5 semantik dan teknik tata letak murni menggunakan CSS Flexbox.

---

## 🎨 Fitur UI/UX

- **Desain Neumorphic Premium:** Mengimplementasikan UI yang lembut menggunakan teknik double `box-shadow` tingkat lanjut untuk menciptakan efek tombol timbul (extruded) dan wadah layar yang cekung (inset) secara realistis.
- **Layout Murni CSS Flexbox:** Dibuat 100% menggunakan Flexbox dengan penentuan ukuran berbasis persentase yang cerdas dan pembungkusan otomatis (`flex-wrap`). Dibangun sepenuhnya tanpa bergantung pada CSS Grid.
- **Bentuk yang Responsif:** Tombol standar otomatis mengunci ke bentuk bulat sempurna menggunakan `aspect-ratio`, sedangkan tombol `0` melebar secara horizontal menjadi bentuk kapsul modern.

---

## 🛠️ Teknologi yang Digunakan

- **HTML5:** Struktur semantik yang bersih menggunakan class khusus (`.number`, `.operator`, `.clear`, `.equal`) yang dioptimalkan untuk proses manipulasi DOM JavaScript ke depannya.
- **CSS3:** Pengaturan warna kustom, pelapisan bayangan (shadow layering) tingkat lanjut, dan penyelarasan Flexbox yang kokoh.
